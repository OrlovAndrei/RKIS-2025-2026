## **Функционал**

После команды несколько вариантов использования только в том случае если они предусматривают эти самые варианты, в случае **если у него нет описания но он есть в списке** это все равно значит что **он полностью функционален.**

Так же этот список **не гарантирует** что данная или аналог данной команды **реализован в парсере команд**.

### **task**

- **add**
- **list**
    - Вывести конкретные задания;
    - Вывести все задания активного пользователя;
    - Вывести все задания конкретного пользователя;
- **remove**
    - Удалить одно задание которое начинается на;
    - Удалить одно задание которое заканчивается на;
    - Удалить одно задание которое содержит;
- **search**
    - Вывести все задания активного пользователя которые начинается на;
    - Вывести все задания активного пользователя которые заканчивается на;
    - Вывести все задания активного пользователя которые содержит;
    - Вывести все задания всех пользователей которые начинается на;
    - Вывести все задания всех пользователей которые заканчивается на;
    - Вывести все задания всех пользователей которые содержит;
- **show**

### **profile**

- **add**
- **change**
    - Сменить на аккаунт который начинается на;
    - Сменить на аккаунт который заканчивается на;
    - Сменить на аккаунт который содержит;
- **list**
    - Показать все профили;
    - Показать количество заданий у каждого пользователя;
- **remove**
    - Удалить один аккаунт который начинается на;
    - Удалить один аккаунт который заканчивается на;
    - Удалить один аккаунт который содержит;
- **search**
    - Вывести всех пользователей которые начинаются на;
    - Вывести всех пользователей которые заканчиваются на;
    - Вывести всех пользователей которые содержат;
- **show**
    - Показать конкретный профиль
    - Показать активного пользователя

## **Пространства имен**

**ShevricTodo.Authentication** - пространство имен ответвленное за антиципацию пользователя и шифрование.

**ShevricTodo.Commands.ProfileObj** - пространство имен для реализации логики работы команд ответвленные за пользователей.

**ShevricTodo.Commands.TaskObj** - пространство имен для реализации логики работы команд ответвленные за задания.

**ShevricTodo.Commands** - пространство имен для реализации логики работы команд.

**ShevricTodo.Database** - пространство имен с представлением базы данных.

**ShevricTodo.Input** - пространство имен для ввода пользователем информации через терминал.

**ShevricTodo.Parser** - пространство имен для обработки команд введенных пользователем.

**ShevricTodo.Parser.Verb** - пространство имен с представлением команд.
