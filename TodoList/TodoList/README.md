# TodoList Console App

Консольное приложение для управления списком задач.

## Возможности

- add <текст> — добавить задачу  
- view — показать список задач  
- done <индекс> — отметить задачу выполненной  
- delete <индекс> — удалить задачу  
- update <индекс> "новый текст" — изменить текст задачи  

## Хранение данных

Используются три массива одинаковой длины:

- todos — текст задач
- statuses — логический статус (выполнено / нет)
- dates — дата последнего изменения

Все массивы расширяются автоматически при необходимости.
