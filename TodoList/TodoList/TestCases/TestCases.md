### ТС-01. Создание нового пользователя

**Описание:**
Проверка успешного создания нового пользователя в системе.

**Предусловия:**
Приложение запущено. Пользователь не авторизован

**Последовательность действий:**
1. В главном меню выбрать команду `create user`
2. Ввести имя: `Ivan Petrov`.
3. Ввести пароль: `SecurePass123`.

**Ожидаемый результат:**
- В консоли отображается сообщение об успешном создании пользователя.
- Пользователь `Ivan Petrov` появляется в списке пользователей 
- Данные пользователя сохраняются в файле данных.

**Описание:**
Проверка поведения системы при вводе неверного пароля.

**Предусловия:**
Приложение запущено. Пользователь `Ivan Petrov` создан.

**Последовательность действий:**
1. В главном меню выбрать команду `login`.
2. Ввести имя: `Ivan Petrov`.
3. Ввести пароль: `WrongPassword`.

**Ожидаемый результат:**
