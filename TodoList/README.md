Обновление 8.12

Обновление 14.02.26 (Лабораторная работа по LINQ)

В этой лабораторной работе мы добавили команду search с использованием LINQ. Теперь пользователи могут гибко искать задачи по тексту, дате и статусу. Команда поддерживает фильтрацию (--contains, --starts-with, --ends-with, --from, --to, --status), сортировку (--sort text/date с флагом --desc) и ограничение результатов (--top <n>). Все операции реализованы исключительно через LINQ (Where, OrderBy, Take) без единого цикла для фильтрации.

Также мы провели рефакторинг архитектуры команд: разделили интерфейсы ICommand и IUndo. Теперь команды, изменяющие данные (add, delete, update, status), реализуют оба интерфейса, а неизменяющие (search, view, read, profile, help) — только ICommand. Это сделало систему отмены действий более прозрачной и надежной.

Для удобства вывода результатов поиска мы добавили в TodoList новый конструктор, принимающий готовый список задач, что позволило переиспользовать существующий метод ViewTasks для отображения результатов. Теперь поиск выглядит так же, как и обычный просмотр задач, что делает интерфейс единообразным и понятным.
