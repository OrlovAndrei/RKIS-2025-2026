# TodoList
Консольное приложение на C#.
- Программа запрашивает имя, фамилию и год рождения пользователя.
- Вычисляет возраст и выводит его на экран.
- Добавление, просмотр, обновление и удаление задач
- Отметка задач как выполненных
- Отображение даты создания или изменения задачи
- Расширение массива при добавлении новых задач
- Просмотр профиля пользователя
- Поддерживает команды:
  - `help` — список команд
  - `profile` — выводит информацию о пользователе
  - `add "текст задачи"` — добавление задачи
  - `view` — просмотр всех задач
  - `exit` — выход из программы
- Автоматически расширяет массив задач при добавлении новых.
Новые команды:
- done <idx> — отметить задачу выполненной
- delete <idx> — удалить задачу
- update <idx> "новый текст" — обновить задачу
