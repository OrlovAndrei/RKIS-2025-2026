# Тест-кейсы для TodoList Application

## Введение
Данный документ содержит набор тест-кейсов для проверки функциональности консольного приложения TodoList.  
Все скриншоты выполнения тестов находятся в папке `Screenshots`.

---

## 1. Пользователи и авторизация

### TC-01 Создание нового пользователя
**Описание:** Проверка успешного создания нового профиля.  
**Предусловия:** Приложение запущено, нет активного пользователя.  
**Последовательность действий:**
1. При запросе "Войти в существующий профиль? [y/n]" ввести `n`.
2. Ввести логин: `testuser`.
3. Ввести пароль: `pass123`.
4. Ввести имя: `Тест`.
5. Ввести фамилию: `Тестов`.
6. Ввести год рождения: `2000`.
**Ожидаемый результат:**  
- Сообщение "Профиль создан! Добро пожаловать, Тест Тестов!"  
- Профиль сохраняется в файле `data/profiles.csv`.
